# Online Restaurant System

----

## Для локального запуска

```
./mvnw package
java -jar target/*.jar
```

Бэкенд здесь: http://localhost:8080/

### Стек технологий
- Spring Boot (Maven)
- MVC Architecture
- Spring Security
- Spring Data JPA
- JWT Authentication
- MySQL
- H2 Database Engine
